# sdas
<b>the station of data staistic system to guangzhou mobile</b><br>
<small>基于r语言和大数据等技术实现对移动工单的验证，通过海量数据的分析显示对未来时刻的预警</small>
<small>本系统作为上述算法验证的演示系统</small>
<h1>introduce</h1>
1.工单验证<br>
2、预警<br>
